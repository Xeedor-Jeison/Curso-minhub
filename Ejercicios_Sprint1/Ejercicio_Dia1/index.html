Bienvenidos a HAMBURGUESAS EL BARRIO

Para nosotros es un placer dar a mostrar nuestra hamburguesa la cual con mucho esfuerzo fue fundada para satifacer sus antojos, hemos recreado una de las mejores recetas para variar un poco el mismo sabor común de las hamburguesas.
